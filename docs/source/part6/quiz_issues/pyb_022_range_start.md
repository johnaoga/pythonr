### Question

Que retourne `list(range(2, 5))` ?

### Options (for single/multiple choice only)

[2, 3, 4, 5]
[2, 3, 4]
[3, 4, 5]

_No response_

### Case Sensitive (text type only)

### Explanation (optional)

`range(2, 5)` génère les nombres de 2 à 4 (la borne supérieure est exclue): [2, 3, 4].
