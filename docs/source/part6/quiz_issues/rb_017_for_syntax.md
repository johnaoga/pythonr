### Question

Quelle est la syntaxe correcte pour une boucle for en R ?

### Options (for single/multiple choice only)

for i in 1:5 { print(i) }
for (i in 1:5) { print(i) }
for i = 1 to 5 { print(i) }
for (i = 1; i <= 5; i++) { print(i) }

### Answer

1

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

R utilise `for (variable in sequence) { ... }`. Exemple: `for (i in 1:5) { print(i) }`.
