### Question

Quelle fonction R utilise-t-on pour charger le package ggplot2 ?

### Options (for single/multiple choice only)

_No response_

### Answer

### Explanation (optional)

`library(ggplot2)` charge le package ggplot2. On peut aussi utiliser `require(ggplot2)` mais `library()` est préféré.
