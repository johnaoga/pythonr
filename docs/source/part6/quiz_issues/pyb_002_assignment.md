Quel opérateur utilise-t-on pour assigner une valeur à une variable en Python ?

### Explanation (optional)

En Python, l'opérateur `=` est utilisé pour l'assignation. `==` est pour la comparaison d'égalité, `<-` est utilisé en R, et `:=` n'est pas un opérateur d'assignation standard.
