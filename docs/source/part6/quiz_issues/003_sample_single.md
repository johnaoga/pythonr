Quelle structure de données est **mutable** en Python ?

### Options (for single/multiple choice only)

### Explanation (optional)

Les listes (`list`) sont mutables en Python, ce qui signifie qu'on peut modifier leurs éléments après création. Les tuples, strings et frozensets sont immuables.
