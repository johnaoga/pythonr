### Question

Quelle syntaxe crée une liste en Python ?

### Options (for single/multiple choice only)

(1, 2, 3)
{1, 2, 3}
[1, 2, 3]
<1, 2, 3>

### Answer

2

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

Les crochets `[]` créent une liste. `()` crée un tuple, `{}` crée un set ou dict.
