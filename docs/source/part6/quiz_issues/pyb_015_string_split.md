### Question

Que retourne `"a-b-c".split("-")` en Python ?

### Options (for single/multiple choice only)

"abc"
["a", "b", "c"]
("a", "b", "c")
["a-b-c"]

### Answer

1

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

`split("-")` divise la chaîne à chaque tiret et retourne une liste des parties: `["a", "b", "c"]`.
