### Question

Quelle fonction R teste si une variable est numérique ?

### Options (for single/multiple choice only)

isnumeric()
is.numeric()
type() == "numeric"
class() == "numeric"

### Answer

### Explanation (optional)

`is.numeric(x)` retourne `TRUE` si x est numérique. R a aussi `is.integer()`, `is.character()`, `is.logical()`.
