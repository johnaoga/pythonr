### Question

Quel est le résultat de `10 %/% 3` en R ?

### Options (for single/multiple choice only)

_No response_

### Answer

3

### Tolerance (number type only)

0

### Case Sensitive (text type only)

### Explanation (optional)

L'opérateur `%/%` effectue une division entière. 10 ÷ 3 = 3.333..., partie entière = 3.
