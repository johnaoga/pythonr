### Question

Comment crée-t-on un entier (integer) en R plutôt qu'un numérique ?

### Options (for single/multiple choice only)

### Answer

### Explanation (optional)

Le suffixe `L` indique à R de stocker comme entier. `5L` est un integer, `5` est un numeric.
