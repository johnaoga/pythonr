### Question

Quelle fonction R retourne le type d'une variable ?

### Options (for single/multiple choice only)

_No response_

### Answer

class

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

### Explanation (optional)

`class(variable)` retourne le type de la variable. Exemple: `class(5)` retourne "numeric".
