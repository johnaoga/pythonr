### Question

En R, quel est l'index du premier élément d'un vecteur ?

### Options (for single/multiple choice only)

0
1
-1
Dépend du vecteur

### Answer

### Explanation (optional)

R utilise l'indexation à partir de 1 (contrairement à Python qui commence à 0). `vec[1]` accède au premier élément.
