### Question

Dans ggplot2, quel argument de `geom_bar()` place les barres côte à côte plutôt qu'empilées ?

### Options (for single/multiple choice only)

position = "side"
position = "dodge"
position = "beside"
position = "group"

### Answer

1

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

`position = "dodge"` place les barres côte à côte. `position = "stack"` (défaut) les empile, et `position = "fill"` normalise à 100%.
