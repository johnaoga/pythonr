### Question

Quelle fonction R retourne la longueur d'une chaîne de caractères ?

### Options (for single/multiple choice only)

length()
len()
nchar()
strlen()

### Answer

2

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

`nchar("Hello")` retourne 5. `length()` retourne le nombre d'éléments dans un vecteur, pas la longueur d'une chaîne.
