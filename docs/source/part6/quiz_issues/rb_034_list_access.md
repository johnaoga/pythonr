### Question

Comment accède-t-on à l'élément nommé "moyenne" dans une liste R ?

### Options (for single/multiple choice only)

liste["moyenne"]
liste.moyenne
liste$moyenne
liste->moyenne

### Answer

2

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

`$` accède aux éléments nommés d'une liste. `liste$moyenne` retourne la valeur associée à "moyenne".
