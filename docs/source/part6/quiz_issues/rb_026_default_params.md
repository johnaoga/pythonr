### Question

Comment définit-on une valeur par défaut pour un paramètre de fonction en R ?

### Options (for single/multiple choice only)

function(x default 5)
function(x: 5)
function(x = 5)
function(x <- 5)

### Answer

2

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

_No response_

### Explanation (optional)

On utilise `=` dans la définition: `function(base, exposant = 2)`. Si non fourni, `exposant` vaudra 2.
