### Question

Quelle fonction Python retourne le type d'une variable ?

### Options (for single/multiple choice only)

_No response_

### Answer

type

### Tolerance (number type only)

_No response_

### Case Sensitive (text type only)

false

### Explanation (optional)

`type(variable)` retourne le type de la variable. Exemple: `type(5)` retourne `<class 'int'>`.
